¿Qué es HTML?

HTML es un lenguage de marcado que se utiliza para representar documentos Web. HTML es la abreviación utilizada para Hyper Text Markup Language

Un lenguaje de marcado es un conjunto de marcas, etiquetas o tags. Así como en un documento Word podemos agregar formato (negritas, cursivas, anchos de marges) en un documento HTML ese formato se lo expresamos con las etiquetas o marcas, cada una de ellas describe un contenido diferente para el documento.

Un documento HTML por lo general es leído por los navegadores Web y desplegado al usuario. Los navegadores no muestran las etiquetas, solo muestan el formato que éstas le dan al documento.

Todo documento HTML esta compuesto por una estructura básica que sirve para que sea reconocido por los navegadores y debe ser respetada. Esta compuesta por:

  • Declaración de la versión del documento, DOCTYPE.
  • Declaración del tipo de doucmento HTML.
  • Cabecera HEADER.
  • Cuerpo del documento BODY.

Un navegador web solo mostrará al usuario el area del cuerpo del documento body.

Elementos HTML

Un documento HTML esta compuesto por elementos HTML, Un elemento esta compuesto por una etiqueta de inicio, una etiqueta de cierre y por el contenido del elemento entre ellas.

Ejemplo:

<p> <--- Etiqueta de inicio
Este es el contenido del elemento
</p> <--- Etiqueta de cierre

Las etiquetas deben ser escritas en minusculas. La documentación de HTML no exige la escritura de las etiquetas en minuscula, pero es recomendado para el caso de validaciones estrictas o documentos XHTML.

Elementos Anidados

Los elementos HTML pueden anidarse, es decir, un elemento puede contener otros elementos. Los documentos HTML son una composición de elementos anidados.

Ejemplo:

<div>
    <p>
        Alguna información relevante
    </p>
</div>

Es importante que nunca olvides la etiqueta de cierre de los elementos que si la posean. Esto ayuda a los navegadores a reconocer la estructura de tu documento.

Elementos vacíos

En HTML un elemento vacío se define como uno que no requiere contenido interno y por lo tanto no posee una etiqueta de cierre. Estos elementos por lo general se cierran con / al final de la etiqueta de inicio.

Ejemplo:

Elemento para dejar una linea <br/>

Si bien HTML5 no exige que se coloque la barra / para cerrar las etiquetas vacías, es recomendable que se coloque en el caso de que quieras pasar validadores de documentos XML

Atributos

Todos los elementos HTML contienen atributos, estos son empleados para agregar información adicional a los elementos. Los atributos son definidos dentro de la etiqueta de inicio de cada elemento HTML. Un atributo es un conjunto de clave/valor.

Ejemplo:

<div title="Un elemento HTML">

Recuerda escribir los atributos en minusculas, HTML no lo exige pero es una buena practica.

Generalmente el valor de un atributo se escribe dentro de comillas. Utilizar comillas dobles o simples queda a juicio del desarrollador y depende del caso en el que se utilicen. Un ejemplo particular del uso de comillas simples o dobles, es cuando el valor del atributo tiene comillas:

<div title="Me dicen 'Tag' o 'Marca'"></div>
<div title='Me dicen "Tag" o "Marca"'></div>

Atributos relevantes

  • id: Este atributo se utiliza como identificador unico de un elemento, siempre debe estar presente en el caso de utilizar un validador de documentos HTML/XHTML.
  • title: Se utiliza para especificar el contenido de una sección como un tooltip. Se activa al mantener el cursor del ratón sobre el elemento.
  • alt: Se utiliza para especificar un texto en el caso de que el elemento HTML no pueda mostrarse.
  • lang: Declara el lenguaje a ser utilizado en un documento HTML. Se utiliza en el elemento de apertura <html>.

Es importante especificar el atributo lang en todo documento HTML, ya que ayuda a regionalizar tu Web en los buscadores.

<!DOCTYPE>

Esta declaración es utilizada para ayudar a los navegadores a mostrar correctamente el documento HTML a los usuario. Actualmente hay varias versiones de documentos HTML por lo tanto hay que informar al navegador que tipo y versión utiliza el documento que queremos mostrar.

Tipo de declaraciones

Actualmente podemos especificar 3 tipo de declaraciones para que un documento HTML sea reconocido.

Ejemplo:

HTML5
<!DOCTYPE html>
HTML 4.0
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
XHTML 1.0
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
